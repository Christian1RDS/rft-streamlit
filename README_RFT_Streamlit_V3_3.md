# RFT Automático — V3.3

Versão com foco em gestão visual e acompanhamento executivo.

## Novidades da V3.3

- Meta de RFT configurável no site
- Alertas visuais quando o RFT fica abaixo da meta
- Comparação entre a carga atual e a carga anterior
- Exportação do snapshot executivo em Excel
- Mantém tudo o que já existia na V3.2:
  - layout aprimorado
  - filtros extras
  - ranking de anomalias e modelos
  - exportação das WOs filtradas
  - histórico melhor organizado
  - gráfico de tendência

## Como rodar

```bash
pip install -r requirements.txt
streamlit run app_rft_streamlit_v3_3.py
```

## Arquivos necessários

- `app_rft_streamlit_v3_3.py`
- `requirements.txt`
