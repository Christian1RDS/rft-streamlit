# RFT Automático — V5.3

Versão focada em **design mais bonito** e **compartilhamento por snapshot**.

## O que a V5.3 traz

- visual mais refinado e moderno
- QG09 principal e QG07 opcional
- anos 2025/2026
- visão diária, semanal, mensal, anual e YTD
- base local SQLite simples
- **exportar/importar snapshot** em JSON para compartilhar manualmente a base entre pessoas
- armazenamento no navegador apenas para filtros leves (ano, posto, semana, mês e modo)

## Como rodar

```bash
pip install -r requirements_v5_3.txt
streamlit run app_rft_streamlit_v5_3.py
```
