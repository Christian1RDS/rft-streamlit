
# RFT Automático — V3.5

Versão com correções de regra visual da meta, calendário funcional e processamento completo de todos os períodos da planilha.

## Novidades da V3.5

- Meta corrigida: **menor ou igual à meta fica verde**; **acima da meta fica vermelho**
- Calendário funcional com botão para **reprocessar a base atual com as regras de calendário**
- Processamento completo da base atual em tabelas automáticas de:
  - Diário
  - Semanal
  - Mensal
  - Anual
  - YTD
- Mantém base histórica separada e ano de comparação configurável

## Como rodar

```bash
pip install -r requirements.txt
streamlit run app_rft_streamlit_v3_5.py
```
